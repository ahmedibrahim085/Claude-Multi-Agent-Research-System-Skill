# Deep Research Command

Comprehensive analysis with maximum coverage.

## Usage

```
/research-deep <topic>
```

## What This Does

1. **Decomposition**: Breaks topic into 4 subtopics (comprehensive)
2. **Parallel Research**: Spawns 4 specialized researchers
3. **MCP Tools**: Leverages all available search engines
4. **Synthesis**: Creates detailed analytical report
5. **Time**: ~25-30 minutes

## When To Use

- Complex, multifaceted topics
- Strategic decision-making
- Academic research
- Market analysis
- Technology assessments

## Example

```
/research-deep cryptocurrency regulation global landscape
```

This might decompose into:
1. Regulatory frameworks by region
2. Technology implications
3. Market impact analysis
4. Future policy directions

## Benefits

✅ Maximum coverage  
✅ Multiple perspectives  
✅ Deeper analysis  
✅ Cross-cutting insights  

## Trade-offs

⚠️ Longer completion time  
⚠️ More researcher coordination
