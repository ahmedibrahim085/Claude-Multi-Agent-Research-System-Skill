# Quick Research Command

Faster research workflow for time-sensitive queries.

## Usage

```
/research-quick <topic>
```

## What This Does

1. **Decomposition**: Breaks topic into 2 subtopics (faster)
2. **Parallel Research**: Spawns 2 researchers
3. **MCP Tools**: Uses available search servers
4. **Synthesis**: Creates focused report
5. **Time**: ~10 minutes

## When To Use

- Time-sensitive queries
- Preliminary investigation
- Quick overviews
- Breaking news research

## Example

```
/research-quick AI safety regulations 2025
```

## Trade-offs

✅ Faster completion  
✅ Less researcher overhead  
⚠️ Less comprehensive coverage  
⚠️ Fewer perspectives
